# Prompt Lookup

This skill provides access to a vast library of community-curated AI prompts, acting as a prompt search engine to improve task execution by using proven patterns.

## Usage Guidelines

When a task requires a specific or complex prompt structure (e.g., writing a marketing email, analyzing a CSV, or specialized coding), use this skill to retrieve an appropriate template.

1. **Search for Prompts**: Use the browser to search `mcpmarket.com` or other prompt repositories for the specific task.
2. **Retrieve and Adapt**: Once a suitable prompt is found, adapt it to the current user's specific context and data.
3. **Apply Best Practices**: Prioritize prompts that are highly rated or frequently used by the community.

## Examples

### Marketing Email
- **User Request**: "Write a cold email for my new SaaS product."
- **Skill Action**: Look up "cold email SaaS" on prompt repositories, find a high-performing template, and fill in the product details.

### Data Analysis
- **User Request**: "Analyze this CSV file for trends."
- **Skill Action**: Look up "CSV data analysis trends" to find a structured prompt that guides the AI through systematic data exploration.

## Limitations

- **Niche Tasks**: May not have specific prompts for extremely niche or brand-new domains.
- **Quality Variance**: Community prompts vary in quality; always review and refine the retrieved prompt before final use.
- **Context Blindness**: The skill provides general templates and does not have inherent knowledge of the user's private data unless provided in the request.

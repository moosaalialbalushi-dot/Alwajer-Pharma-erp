# Brainstorming Skill

This skill helps facilitate a high-volume idea generation process followed by a structured evaluation phase, based on the principles of Alex Osborn and IDEO.

## Core Workflow

Brainstorming is divided into two distinct phases to prevent premature criticism from stifling creativity.

### Phase 1: Idea Generation (Divergent Thinking)

The goal is **quantity over quality**. Follow these rules:
1.  **Defer Judgment**: No criticism or evaluation allowed.
2.  **Encourage Wild Ideas**: The more unconventional, the better.
3.  **Build on Others**: Use "Yes, and..." to expand on existing thoughts.
4.  **Stay Focused**: Keep the session within a set time limit (e.g., 10-20 minutes).

**Actionable Steps:**
- Define the problem clearly before starting.
- Record every idea immediately without filtering.
- If the flow slows down, suggest a new angle or perspective.

### Phase 2: Idea Evaluation (Convergent Thinking)

Once the generation phase ends, shift to refining and organizing the most promising concepts.

**Actionable Steps:**
1.  **Organize**: Group similar ideas into themes or categories.
2.  **Discuss**: Briefly clarify any ambiguous ideas.
3.  **Refine**: Combine or improve ideas based on feasibility and impact.
4.  **Select**: Use voting or a matrix to identify actionable solutions.
